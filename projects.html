Design of a mind controlled hand prosthesis using Machine Learning

This project combines various technologies and components to create a complete hand prosthesis. It integrates the mechanical Cyborg Beast hand with a CAD-designed forearm, making it accessible for home development using Arduino components. The hand's movements are controlled by tensor cables attached to step motors. Python is used for software development and data analysis, while OpenVibe is employed for processing EEG signals. EEG signals, captured from the user's brain activity, are utilized to develop a brain-computer interface. The LDA machine learning algorithm achieves high accuracy with 84.7% in interpreting the user's intentions based on the EEG data. .
